# 📝 System Prompts (`prompts`)

This directory contains system prompt guides that dictate the behavior and structure of AI responses when CommiX acts as a Git commit message generator.

## 📂 File Summary & Prompt Templates

| File | Type | Description |
| :--- | :--- | :--- |
| [`gitmoji_prompt.md`](file:///d:/open-source/commix/prompts/gitmoji_prompt.md) | **Emoji Conventional Prompt** | A specialized prompt that instructs the AI model to generate conventional commit messages prefixed with standard Git emojis (e.g., ✨ for feature, 🐛 for fix). |
| [`sample_prompt.md`](file:///d:/open-source/commix/prompts/sample_prompt.md) | **Standard Conventional Prompt** | A baseline prompt instructing the AI to output strictly structured, conventional, and plain text semantic commit messages complying with standard specifications. |
