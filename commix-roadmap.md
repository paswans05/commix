# 🚀 Roadmap

CommiX is growing fast. Here’s the official roadmap outlining what’s coming next, what’s in progress, and what’s planned for future releases.

## 🔥 **Phase 1 — GitSmart Mode**

- [x] Auto-detect staged files
- [x] AI-generated commit messages
- [x] Support for Conventional Commits
- [x] Support for Emoji Commits
- [x] One-click “Generate Commit” button
- [ ] Terminal sidebar commit icon
- [ ] Commit history improvements

## ✅ **Phase 2 — Core (MVP)**

- [x] AI Edit (Improve, Fix, Optimize)
- [x] AI Explain Code
- [x] Selection-based actions
- [x] Sidebar Panel (CommiX View)
- [x] Status Bar Toggle (⚡ ON / OFF)
- [x] Settings Page (API Key, Model Select)
- [x] Terminal Icon Support (left sidebar integration)

## 🎯 **Phase 3 — AI Edit Mode (Signature Feature)**

- [x] Function-level icons
- [x] Improve / Fix / Refactor / Document buttons
- [x] Convert code (JS → TS, JS → Python, etc.)
- [x] Diff-based preview
- [x] Inline transformation suggestions
- [x] Smart patch apply

## ⚙️ **Phase 4 — Model Engine Upgrade**

- [x] OpenAI models
- [x] Gemini
- [x] Groq
- [x] NVIDIA NIM SDK

- [x] Token usage & benchmarking

## 🎨 **Phase 5 — CommiX Studio**

- [x] AI Snippet Generator
- [x] AI Test Generator
- [x] AI Documentation Builder
